# ft_ls

Syntax: ./ft_ls [-lRartG] [file [...]]

FLAGS:
* l  : Show the long version of the output.
* R  : Recurse into directories.
* a  : Show EVERYTHING, that includes hidden files.
~~* t  : Sort the files based on their timestamp.~~ (that didn't work so well)
* G  : Colors!
